교과서 소개

각종 IT 서적과 이공계 대학 교재 등을 출간하며 IT 출판 분야를 선도해 온 한빛출판네트워크에서 만든 교과서입니다.

학생들의 수준과 학습 내용 간의 관계를 최우선으로 생각하여 '흐름이 보이는 교과서', '구성이 탄탄한 교과서'를 지향합니다.

  • 1. 도입

    본문 내용과 관련된 실생활의 사례를 삽화로 보여 주고 이를 바탕으로 생각할 문제를 제시하여 학생들의 흥미를 유발합니다.

    화면 크게보기

  • 2. 전개

    핵심 개념을 삽화나 도식을 이용하여 소개하고 다양한 <활동>을 통해 확인해 봅니다. 학습 진도가 빠른 학생에게는 <심화 활동>과 <하나 더 알기> 등을 제시하여 수준별 학습을 진행합니다.

    화면 크게보기

  • 3. 정리

    본문에서 배운 내용을 다른 상황에 적용해 보거나 친구들과 협력하여 해결하는 문제를 풀어 봅니다. 문제를 푼 후에는 마인드맵을 보고 본문에서 배운 핵심 용어와 개념을 정리합니다.

    화면 크게보기

  • 4. 평가

    문제 해결 프로젝트, 마인드맵 채우기, 단답형/서술형 문제 등 다양한 유형의 문제를 통해 컴퓨팅 사고력과 협력적 문제 해결력을 키웁니다.

    화면 크게보기

교과서 주요 특징

실생활 사례 중심

매 단원마다 실생활의 다양한 사례를 제시하여 학생들이 쉽게 공감하고 학습할 수 있습니다.

문제 해결 활동 중심

친구들과 협력하며 문제를 해결하는 활동 중심의 교과서입니다.

흐름이 보이는 목차 구성

학습 내용 및 순서가 한눈에 들어옵니다. 현재 어디까지 공부했고 앞으로 배울 내용이 무엇인지 쉽게 알 수 있습니다.

짜임새 있는 내용 구성

단원 간 학습 내용을 연계했으며, 문제 해결을 위한 수행 과정이 포트폴리오 형태로 누적됩니다.

교과서 미리보기 지도서 미리보기 강의교안 • 샘플

교과서에 사용된 피지컬 컴퓨팅 도구(교과서용 키트)는 비트브릭 홈페이지에서 구매하실 수 있습니다.
